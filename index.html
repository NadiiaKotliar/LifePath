О нашем учебном проекте

Идея

Этот сайт создан как учебный проект для практики HTML, CSS и JavaScript. Все элементы кода открыты для изучения.

Технологии

Чистая семантическая верстка, адаптивный дизайн, современный CSS (Flexbox/Grid). Без использования CMS.

Обучение

Цель проекта — освоить на практике создание современных веб-сайтов с нуля. Каждый раздел — это новый навык.

Этапы создания сайта

1

Планирование и дизайн

Прототипирование в Figma, подбор цветовой палитры и шрифтов, создание структуры.

2

Верстка

Семантическая разметка HTML, стилизация с помощью CSS, создание адаптивного дизайна.

3

Программирование

Добавление интерактивности с помощью JavaScript: меню, формы, анимации.

4

Публикация

Размещение на GitHub Pages, настройка домена, оптимизация скорости загрузки.
